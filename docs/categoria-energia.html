El Diario le brinda al lector un mapa del ecosistema noticiero, y no aspira a la supuesta 'obejtividad' — sino que ayuda a crear el contexto necesario que algunos medios no proveen, ya sea por accidente o con intención, para entender las dinámicas de poder dentro del capitalismo colonial que vive Puerto Rico.

El Diario está disponible en la web abierta, y se actualiza una vez al día.
